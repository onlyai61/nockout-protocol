# NOCKOUT Unity UI Prototype

UI-only Unity prototype for the NOCKOUT Protocol landscape mobile flow.

## Contents

- `Assets/NockoutUIPrototype/NockoutUIPrototype.unity`
- Runtime UI builder script for combat bot + 3 support bots
- Support bot customization modals
- Ranking modal concept
- Light/Dark toggle
- Arena background and robot image resources

## Import

Use one of these options:

1. Copy this folder's `Assets/NockoutUIPrototype` into a Unity project.
2. Or import `package-downloads/nockout_ui_prototype.unitypackage`.

Then open:

```text
Assets/NockoutUIPrototype/NockoutUIPrototype.unity
```

If the scene needs to be regenerated, use:

```text
NOCKOUT > Create UI Prototype Scene
```

## Scope

This is not a playable combat build. It is only for validating:

- Combat bot + support bot loadout UI
- Up to 3 support/maintenance bots
- Ranking screen direction
- Mobile landscape layout density
- Modal-based detail flow

Recommended Game view size: `844 x 390`.
